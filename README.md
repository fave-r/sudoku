sudoku
======

Resolveur de sudoku by odet_a & fave_r

partie 1 : Résous une seule grille

Les grilles doivent etre placees dans un fichier texte comme ceci :

|------------------|

| 2         4   6 9|

| 1         6   4  |

|   4         7    |
|         3   8 5  |
| 8 1     2        |
| 9                |
|     7 1         2|
|           2 3   4|
| 5   2            |
|------------------|
